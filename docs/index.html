Scores what usually breaks MCP installs.

Reliability

Find parse errors, missing commands, dead local paths, broken environment references, unsupported transports, and stale package metadata.

Security

Flag inline secrets, secret-like args, broad filesystem mounts, private repo leaks, and risky values before a client loads them.

Context Hygiene

Estimate loaded tool count, highlight duplicate servers, and show which configs are adding noise to your agent context.

Use it as a CLI or as an MCP server.

Run a one-off local scan, export a redacted Markdown or HTML report, or install MCP Doctor into Codex, Claude Code, Cursor, Windsurf, GitHub Copilot, VS Code, Cline, Roo Code, Continue, or Zed as a normal stdio MCP server.

npx @inferensys/mcp-doctor scan --workspace .
npx @inferensys/mcp-doctor report --format html
npx @inferensys/mcp-doctor server

codex mcp add mcp-doctor -- \
  npx -y @inferensys/mcp-doctor server
